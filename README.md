### **Kurulum Adımları:**

1.  Projeyi bilgisayarınıza indirin:
    
    ```bash
    git clone https://github.com/aliriza1926/Bakery_Menu  
    
    ```
    
2.  Proje klasörüne gidin:
    
    ```bash
    cd Bakery_Menu  
    
    ```
    
3.  Gerekli paketleri yükleyin:
    
    ```bash
    flutter pub get  
    
    ```
    
4.  Uygulamayı başlatın:
    
    ```bash
    flutter run  
    
    ```
